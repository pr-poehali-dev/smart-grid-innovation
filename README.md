# Horizon Adventures

![Preview](https://cdn.poehali.dev/templates/horizon-adventures/preview-ru.jpg)

Лендинг для туристической компании, организующей экспедиции в пещеры. Темный дизайн с эффектами glassmorphism, полноэкранный hero-блок с фоновым изображением, секции преимуществ, пошаговый маршрут экспедиции, FAQ-аккордеон, контактная форма и футер с подпиской на рассылку.
